# The Practice — a local movement learning system

A self-contained 12-week movement foundations program. No build step, no server,
no internet connection required.

## Live version

**https://pierrechass.github.io/movements/** — hosted for free on GitHub Pages,
works on any device (including phones) with an internet connection.

## How to use

Open **`index.html`** in any web browser (double-click it). Everything else is
linked from there.

## To send it to another laptop

Copy the whole folder (or zip it) and open `index.html` on the other machine.
All links between pages are relative, so it works from any location — a USB
stick, a Downloads folder, anywhere.

## Files

| File | What it is |
|------|------------|
| `index.html` | Hub — philosophy, skill-tree roadmap, glossary, journal templates. **Start here.** |
| `exercises.html` | Exercise & skill library — every movement in detail |
| `month1.html` | Weeks 1–4 — Foundations |
| `month2.html` | Weeks 5–8 — Locomotion & Transitions |
| `month3.html` | Weeks 9–12 — Integration |

## Languages

Every page is also available in French and German, using the same filename with a
`-fr` or `-de` suffix (e.g. `index-fr.html`, `exercises-de.html`, `month1-de.html`).
A language switcher (EN / FR / DE) in the top bar of every page links to the
matching page in each language. Exercise/skill proper nouns (Beast, Crab, Hollow
Body, CARs, etc.) are kept in English in all languages; only the surrounding
instructional text is translated.

## Notes

- Pages follow your OS light/dark setting automatically; the ◐ button
  (bottom-right) toggles it manually and remembers your choice.
- Designed for ~2–3 sessions/week at ~40 min. Print any page (they're
  print-friendly) if you'd rather train from paper.
